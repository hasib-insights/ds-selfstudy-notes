### Python Numbers  
There are three numeric types in Python:  
1. int
2. float
3. complex
Example
```python   
x = 1    # int
y = 2.8  # float
z = 1j   # complex
```
To verify the type of any object in Python, use the type() function:
```python
print(type(x))
print(type(y))
print(type(z))
```
out:
```
<class 'int'>
<class 'float'>
<class 'complex'>
```
